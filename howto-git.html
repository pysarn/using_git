Git

ฝึกใช้ git ของ codeschool 
git config --global user.email "you@example.com"
    set user 
git init
    Initialized empty Git repository เป็นคำสั่งเริ่มต้นใช้งาน git โดยจะสร้างโฟลเดอร์ .git ขึ้นมาซึ่งเป็น hidden folder เพื่อใช้เก็บข้อมูลต่างๆ ของ Git repository 
git status
    คำสั่งที่ใช้ดูสถานะ 
git add <file>
    คำสั่งเพิ่มไฟล์ หรือโฟลเดอร์เข้าไปยัง Staging Area 
git add '*.txt'
    เพิ่มครั้งละหลาย ๆ ไฟล์ด้วยการใช้ wildcard - Don't forget the quotes! 
git rm --cached <file>
    นำไฟล์ออกจาก Staging Area 
git diff --staged
    ตรวจสอบว่ามีไฟล์ใหม่ add เข้ามามั๊ย 
git reset­ <file name>
    เป็นคำสั่งสำหรับการยกเลิก staged ของไฟลื ที่เราต้องการออกไป ก่อนที่จะ commit 
git commit -m 'comment message'
    ทำการคอมมิทไฟล์ให้เข้า หรือออกจาก local repository 
git log และ git log --sum­mary
     เป็นคำสั่งเพื่อดูประวัติการ commit ต่างๆ ใน git repository 
 git remot­e add origin https://github.com/username/repository_name.git
     เป็นคำสั่งใช้สำหรับการเพิ่ม remote repository ขึ้นมาชื่อว่า origin เพื่อนำมาใช้งานต่อไป 
 git push -u origi­n maste­r
     เป็นคำสั่งส่งไฟล์และโฟลเดอร์จาก local repository ไปยัง remote repository (github.com) 
 git pull origi­n maste­r
    ทำการดึงไฟล์และโฟลเดอร์ต่าง ๆ ที่มีการเปลี่ยนแปลง (จากคนอื่นทำ) ลงมาที่เครื่องของเรา 
git diff HEAD
    Differences - ตรวจสอบว่ามีการเปลี่ยนแปลงอะไรที่แตกต่างจากที่เราคอมมิทไปบ้าง what is different from our last commit
